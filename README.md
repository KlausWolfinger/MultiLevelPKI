# Installation Steps

### Install git client

1. apt-get install git

### Clone the PKIT master branch
2. clone the master
3. git clone https://github.com/KlausWolfinger/PKIT.git
4. cd PKIT

### Set x flag
5. chmod u+x pki.sh

### Launch the PKIT
6. ./pki.sh
